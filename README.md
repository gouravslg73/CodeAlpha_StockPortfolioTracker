# CodeAlpha_StockPortfolioTracker

## 📌 Project Description

This project was developed as part of the **CodeAlpha Python Programming Internship**.

The Stock Portfolio Tracker is a Python-based application that allows users to manage stock investments, record transactions, calculate portfolio values, and generate reports. The project uses Object-Oriented Programming (OOP), file handling, and JSON files for data storage.

---

## ✨ Features

- User Authentication
- Add and Manage Stock Portfolio
- Buy and Sell Stock Records
- Portfolio Value Calculation
- Generate Investment Reports
- Store Data Using JSON Files
- Error Handling
- Modular Python Code

---

## 🛠 Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- JSON
- File Handling
